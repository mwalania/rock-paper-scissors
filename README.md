# rock-paper-scissors
This is a simple rock paper scissors(RPS) game completed as part of **The Odin Project** Foundation Curriculum.
The goal of the project was o build a rps game that runs on the console using HTML and JavaScript.

## Technologies used
- HTML
- JavaScript
- Git/Github

## Learning Outcomes
By the end of the project i equipped myself with some skills like:
- Declaring functions using `let` and `const`
- Local and Global scope of variables
- Writing/declaring and calling functions
- Using functions parameters
- Using JavaScripts in-built functions like Math e.g *Math.random()*
